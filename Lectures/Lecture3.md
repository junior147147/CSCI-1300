----------------------------------------------

Sum of Numbers
-----------------

```python

x = 12   // "=" is an assignment operater (binary operater with left hand side and right)

x = x + 23 || x += 23 // Python perform the right hand side first, then assign it to x on the left

print (x) 

//35

```
String concatenation
---------------------

```python

a = "hello"
b = "world"

c = a + b

print(c) 

// hello world

```
